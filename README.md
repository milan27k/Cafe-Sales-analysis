# Cafe-Sales-analysis
☕ Cafe Sales Data Analysis & Dashboard

🚀 Beginner-friendly end-to-end Data Analysis project using Python and Power BI.

## 📌 Project Overview
This project focuses on analyzing cafe sales data to uncover meaningful insights such as top-selling products, revenue trends, and category performance.

The project includes:
- Data cleaning and preprocessing using Python
- Interactive dashboard creation using Power BI

## 🛠 Tools & Technologies
- Python
- Pandas
- Power BI
- Jupyter Notebook

## 📂 Dataset Description
The dataset contains cafe sales transaction data with the following fields:
- Item (Product name)
- Category (Product category)
- Quantity (Units sold)
- Price (Per unit price)
- Date (Transaction date)

## 🔍 Data Cleaning & Preprocessing
The dataset was cleaned using Python in Jupyter Notebook:
- Removed missing (null) values
- Eliminated duplicate records
- Converted data types (e.g., Date column)

## 📊 Data Analysis
Key analysis performed:
- Top-selling products based on quantity
- Category-wise revenue analysis
- Total sales calculation

## 📈 Power BI Dashboard
An interactive dashboard was created using Power BI to visualize insights.

### Dashboard Features:
- Total Sales Overview (KPI cards)
- Top Selling Products
- Monthly Revenue Trend
- Sales Trends Visualization
- Filters for better interaction

## 💡 Key Insights
- Salad is highest selling products
- 2nd quarteer Generates Highest Revenue
-June (Month) Records the highest sales in a year
- Helped understand customer demand trends

## 📁 Project Files
- `cleaning_cafe_dataset.ipynb` → Data cleaning & analysis (Python)
